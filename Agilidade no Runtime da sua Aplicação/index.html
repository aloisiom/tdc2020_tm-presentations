Benefícios:

  • RASP: Simples de implantar, não precisa alterar a aplicação
  • Inspeciona todas as ações da aplicação
  • Aplicações Serverless (lambdas...)
  • Ajuda os desenvolvedores a priorizarem o backlog
  • Pode ser uma ação de mitigação para momentos em que é uma decisão de negócio
Application Security
RASP WAF
Accuracy Detection of malicious input only when passed to library calls where exploitation would occur. Monitors inbound and outbound data and logic flows. Detection based on naïve pattern matching, without considering whether the input would be passed to vulnerable code.
Time to Value No need to know locations of existing vulnerabilities in application code; can act as a virtual patch against a vulnerability. Requires extensive testing and confguration to adequately cover the application.
Reliability Will not fail open under high load—code is always instrumented, regardless of server load. Single point of failure; likely to fail open under high load, leaving the formerly protected web application vulnerable
Platforms Any instrumented application. Web applications.
Visibility May provide detailed feedback to developers to show how to remediate code vulnerabilities. Ofers no detailed insight into the application.
Network Protocols Protocol agnostic; handles HTTP, HTTPS, AJAX, SQL and SOAP with equal ease. Must be able to understand the application’s network communication.
Language Coverages Theoretically language agnostic but requires complicated, language-specifc builds—currently known products support .NET and Java. Language agnostic; not bound by programming language type.
